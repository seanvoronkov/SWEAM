About Swimming Points

Understanding how swimming points are calculated

How Points Are Calculated

Swimming points are calculated using the formula: 1000 × (Base Time / Actual Time)³. The base time is derived from the current world record for each event.

World Records

Our calculator uses the latest world records as base times for calculating points, ensuring accurate and up-to-date comparisons.
